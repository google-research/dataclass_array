## Principle

When the user asks you to explore, research, or investigate a topic without a
specific coding goal, you MUST export your findings as a permanant artifact.
Chats are temporary, memory is permanant.

## Steps

1.  **Conduct the research**: Read the provided documents, query the necessary
    systems, and synthesize the information.
2.  **Export using memory**: Use the `use-memory` skill to create a persistent
    record of your findings.
3.  **Provide a summary**: In your chat response, briefly summarize the results
    and inform the user that the full details have been saved to the artifact
    file.
